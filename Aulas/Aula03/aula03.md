### O que é a TAG div?
    Div é uma TAG que cria um bloco generico para agrupar elementos. Existem outras TAGs que podemos agrupar no HTML, porem a DIV é a mais genérica.


### outras exemplos de TAG usadas para agrupar elementos
    div
    nav
    article
    section
    header
    spam
